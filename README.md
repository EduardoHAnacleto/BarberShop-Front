# 🚀 Why this project?

Managing a modern barbershop involves much more than booking appointments.

This application simulates a complete business platform capable of managing customers, employees, appointments, services, analytics and day-to-day operations through a responsive web application.

The project was designed to demonstrate modern Full-Stack development using production-oriented technologies, clean architecture and Docker-based deployment.

# 🏗 Architecture

```text
Browser
   │
 Nginx
   │
 ├── Nuxt 3 Frontend
 └── ASP.NET Core API
          │
      SignalR
          │
     SQL Server
```

---

# 🛠 Technology Stack

| Layer | Technology |
|--------|------------|
| Frontend | Nuxt 3, Vue 3, TypeScript |
| Backend | ASP.NET Core (.NET 8) |
| Database | SQL Server |
| Authentication | JWT + Google OAuth |
| Real-Time | SignalR |
| DevOps | Docker, Docker Compose |
| Reverse Proxy | Nginx |
| Testing | Playwright |
| Monitoring | Sentry |
